{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} This is a simple extension to add support for Into the Breach to the Vortex mod manager. With this extension you can install, remove and manage your Into the Breach mods using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with Subset Games and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your Into the Breach mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/vortex-intothebreach" %}} The extension is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Supports Most Mods" %}} You should be able to install almost any mod for Into the Breach automatically. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
